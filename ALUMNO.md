## Por qué necesitamos Git

Sin Git, cuando trabajamos en equipo...
[Se vuelve muy complicado llevar un control y organizacion, asi como planeacion. Se vuelven complicado coordinar los cambios que se quiere llevar a un proyecto/codigo]

## Los 3 conceptos clave de Git

1. **Repositorio:** Lugar donde se almacena un proyecto o archivos
2. **Commit:** Es como un guardado, se comenta algun cambio en tu archivo y se puede ver desde GitHub
3. **Rama:** Un espacio libre del repositorio donde se trabaja
